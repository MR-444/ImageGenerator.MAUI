    stays hidden — all Pollinations descriptors inherit
    PollinationsDescriptorBase with ImagePrompt:false (line 53). The

    (documents/Pollinations API Reference.html and the v0/gen.pollinations.ai
    /openapi.json it loads). Gap to close:
      1. Flip ImagePrompt:true on the descriptors that actually support
         it (likely a subset — verify per-model in the API docs, not
         blanket on the base class).
      2. PollinationsImageGenerationService.BuildUrl currently only
         emits prompt/model/width/height/seed/enhance/safe. Add image=
         (or whatever the spec calls it) when SelectedImages is non-
         empty. The encoding is the unknown bit: Pollinations is GET-
         only so expects a URL, but the app holds base64. Likely needs
         either a URL-input path (Pollinations may accept data: URIs)
         or a host-the-image-first step.
      3. ModelCapabilities for Pollinations needs MaxImageInputs > 0
         only where supported.

[x] (Moderate) .Result after Task.WhenAll — tuple-await refactor
    Commit: 2d96a6d (2026-05-20)
    The only .Result access in the codebase. After
    `await Task.WhenAll(replicateTask, pollinationsTask)` the reads
    were functionally safe (both tasks complete) but read as
    sync-over-async at a glance. Replaced with
    `var (replicate, pollinations) = (await replicateTask, await
    pollinationsTask);` — both fetches still start concurrently
    (no await until the tuple), parallelism preserved, .Result gone.
    339/339 green.

HOW TO PICK THE NEXT ITEM
-------------------------

1. Prefer Correctness over Cosmetic over Missing-tests.
2. The single Correctness item (.Result refactor) is the smallest
   commit in the queue — do it first.
3. Within Cosmetic, smaller-scope first: seal-bundle (one ~5min
   commit), constant-wire (one ~5min commit), and the second
   /select, fix (one ~5min commit) are all mechanical. DispatchToUi
   extract is ~15min. Suppression-flag move is the largest cosmetic
   and ripples through GeneratorViewModel + InputImagesCoordinator.
4. The Missing-Tests item (Default() registry completeness) pairs
   naturally with the seal-bundle commit since both touch the
   descriptor surface.
5. Update this file when you start (set [~]) and when you finish
   (set [x] + commit hash).
